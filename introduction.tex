\section{Introduction}

Dart (first called Dash)  is a class-based, object-oriented programming language for the web developed by Google and published under the open source ``New BSD License''\footnote{\url{http://www.opensource.org/licenses/bsd-license.php}}.\cite{Wikipedia} %The developers at Google are talented and have shown they are amibitious with their projects. So Dart will likely become a great language challenging browsers, mobiles and servers.

% The source code and issue tracker can be found at the google code page of Dart\footnote{\url{http://code.google.com/p/dart/}}. A thorough language tour and the latest downloads of the provided tools can be found at the offical project webiste\footnote{\url{http://www.dartlang.org/language-tour/}}.

This paper will give a brief introduction to client-side development with Dart by developing a small contact manager written in Dart. Some features and language concepts of Dart will be used and explained.% The evaluation will, among others, discuss browser support and development state.
